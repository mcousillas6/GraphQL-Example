GraphQL Example Project
=======================================================

## Overview
This project is an example of a simple implementation of a GraphQL service with the following features:
- User creation.
- User authentication with JWT.
- Access to private calls using JWT.

## Environment setup
- `Dotenv` is set up to manage environment variables locally, use the `.env.sample` file as a base and use one `.env` file per environment (so, development, test and production).

## Setup
- To run the project in development mode just run `npm run dev`.
- Run `npm start` to run in production mode.
- Run `npm test` to run the test suite.
